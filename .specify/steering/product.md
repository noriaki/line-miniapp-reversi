# Product Overview

LINE Mini App Reversi is a high-performance Reversi (Othello) game that runs natively within the LINE messaging platform, offering AI-powered gameplay through WebAssembly technology.

## Core Capabilities

- **AI-Powered Gameplay**: Leverages Egaroucid WebAssembly engine for intelligent AI opponents with configurable difficulty
- **LINE Platform Integration**: Seamless integration with LINE ecosystem via LIFF SDK, supporting user profiles and LINE-specific UI patterns
- **Touch-First Mobile Experience**: Optimized for smartphone interaction with intuitive tap-to-play mechanics and responsive visual feedback
- **Fast Loading**: Optimized for Vercel Edge deployment with efficient initial load times

## Target Use Cases

**Primary Scenario**: LINE users playing casual Reversi games against AI during commutes or breaks, with zero installation friction (runs directly in LINE app)

**Technical Showcase**: Demonstrates modern web platform capabilities (WebAssembly, Web Workers, static export) within a constrained mini-app environment

## Value Proposition

Combines the accessibility of a browser-based game with near-native performance through WebAssembly, eliminating app store friction while delivering competitive AI gameplay. The LINE integration provides familiar authentication and social context without requiring separate user management.
